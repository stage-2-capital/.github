## What lives here

Each repository in this Org is a Claude Skill — a packaged GTM workflow you can install and run in Claude in about 60 seconds. Every Skill featured in the GTM AI series gets a home here, with the full file you need to install it, plus a guide on how to use it.

Skills are contributed by founders, operators, and revenue leaders across the Stage 2 portfolio and LP network. Each one has been used in production by the operator who built it.

[Browse the Skills →](https://github.com/orgs/stage-2-capital/repositories)

---

## How to use a Skill

1. **Pick a Skill** from the repository list and open its folder.
2. **Read the guide.** Each Skill has its own README with setup steps, prerequisites, and example use cases.
3. **Download the Skill folder.** Click the green **Code** button → **Download ZIP** → unzip → grab the Skill folder.
4. **Install in Claude.** Go to **Settings → Capabilities → Skills** in Claude (web, desktop, or Cowork) and upload the folder.
5. **Trigger the Skill** using the prompts listed in the Skill's guide.
